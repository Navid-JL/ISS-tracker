## Run Locally

Clone the project

```bash
  git clone   git clone https://github.com/Navid-JL/ISS-tracker
```

Go to the project directory

```bash
  cd ISS-tracker/
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
